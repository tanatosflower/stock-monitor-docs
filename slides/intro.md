# stock-monitor 概要

株価・指標の自動監視・通知サービスの概要紹介です。

---

## アジェンダ

1. **サービスの目的**
2. **主な機能**
3. **システム構成の概要**
4. **運用フローのポイント**

---

## サービスの目的

- 株価・財務指標を**定期的に自動取得**し、設定した条件に一致したときに通知します。
- 手動でチャートを確認する手間を省き、見逃しをなくすことが目的です。

---

## 主な機能

| 機能 | 概要 |
| --- | --- |
| データ取得 | 複数の証券 API から株価・指標を収集 |
| 条件評価 | 閾値・移動平均・パターンなど複数ルールを組み合わせ判定 |
| 通知 | Slack / メール / Webhook などへアラート送信 |
| 運用管理 | 監視銘柄・条件の追加・削除・一時停止 |
